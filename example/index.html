jquery-ui-timeslider demo


Allows ctrl-left/right to move the slider one minute step (i.e. it matches the datepicker keyboard interface), and ctrl-shift-left/right to move one hour on the slider. Of course, it's still a slider, so the mouse interface is obviously even simpler, but unlike other popular / less popular alternatives, it doesn't break keyboard navigation.
